# Pokemon AI Project

This project is an AI program designed to navigate through Pokémon FireRed while following Nuzlocke rules.

## Features
- **Automated navigation**: Moves the player through the starting room.
- **Screen capturing**: Takes a screenshot of the emulator screen.
- **Logging**: Keeps track of AI actions and errors.
